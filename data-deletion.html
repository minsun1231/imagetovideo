데이터 삭제 요청

Video Maker에서 사용자 데이터 삭제를 요청하는 방법을 안내합니다. 사용자의 개인정보 보호 권리를 존중하며, 언제든지 데이터 삭제를 요청할 수 있습니다.

삭제되는 데이터

완전 삭제되는 데이터

  • Instagram 계정 연동 정보 (사용자 ID, 사용자명)
  • 앱에서 생성한 동영상 파일
  • 앱 사용 기록 및 설정 정보
  • 기기에 저장된 모든 임시 파일
  • 서버에 저장된 사용자 관련 모든 데이터

Instagram 원본 데이터

중요: Video Maker는 Instagram 원본 데이터를 저장하지 않습니다. Instagram에서 가져온 이미지는 동영상 변환 목적으로만 임시 사용되며, 변환 완료 후 즉시 삭제됩니다.

데이터 삭제 방법

방법 1: 앱 내에서 직접 삭제 (권장)

  1. Video Maker 앱 실행
  2. 설정 메뉴 접속
  3. "계정 관리" 선택
  4. "Instagram 연동 해제" 버튼 클릭
  5. "모든 데이터 삭제" 옵션 선택
  6. 삭제 확인

처리 시간: 즉시 삭제

방법 2: Instagram 설정에서 앱 연동 해제

  1. Instagram 앱 또는 웹사이트 접속
  2. 설정 → 보안 → 앱 및 웹사이트
  3. "Video Maker" 앱 찾기
  4. "제거" 버튼 클릭
  5. 연동 해제 확인

처리 시간: 24시간 이내 자동 삭제

방법 3: 이메일을 통한 삭제 요청

이메일 주소: privacy@videomaker.app

제목: 데이터 삭제 요청

포함 정보:

  • Instagram 사용자명 또는 사용자 ID
  • 삭제 요청 사유 (선택사항)
  • 본인 확인을 위한 추가 정보

처리 시간: 요청 접수 후 3일 이내

삭제 완료 확인

삭제 완료 통지

  • 앱 내 삭제: 즉시 화면에 완료 메시지 표시
  • 이메일 요청: 처리 완료 후 확인 이메일 발송
  • Instagram 연동 해제: Video Maker 앱에서 자동 로그아웃

데이터 삭제 후 복구

주의사항: 한 번 삭제된 데이터는 복구할 수 없습니다.

  • 삭제 전 생성한 동영상 파일을 기기에 저장해두시기 바랍니다
  • 재사용을 원하는 경우 Instagram 계정을 다시 연동해야 합니다
  • 삭제된 데이터는 어떠한 경우에도 복구되지 않습니다

법적 보관 의무

즉시 삭제되지 않는 정보

  • 접속 기록: 정보통신망법에 따라 3개월간 보관
  • 결제 기록: 전자상거래법에 따라 5년간 보관 (해당 시)
  • 분쟁 관련 기록: 분쟁 해결 시까지 보관

데이터 삭제 현황 확인

데이터 삭제 현황을 확인하거나 추가 문의사항이 있는 경우:

연락처

  • 이메일: privacy@videomaker.app
  • 제목: 데이터 삭제 현황 확인
  • 응답 시간: 업무일 기준 3일 이내

관련 링크

  • 개인정보처리방침
  • 서비스 이용약관
  • Instagram 데이터 정책
